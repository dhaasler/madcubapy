##########
MADCUBA IO
##########

The IO package of `madcubapy` allows the user to read MADCUBA products like FITS
and *.spec* files alongside their :ref:`history files <hist_file>`.

MadcubaMap
==========

.. toctree::
    :maxdepth: 1
    :hidden:

    madcubamap

The `~madcubapy.io.madcubamap.MadcubaMap` class stores the data and metadata of
FITS files.

Look into the :ref:`MadcubaMap page <info_madcubamap>` for a detailed overview
of the class.

SpectraContainer
================

.. toctree::
    :maxdepth: 1
    :hidden:

    spectracontainer

The `~madcubapy.io.spectracontainer.SpectraContainer` class stores spectra data
exported by MADCUBA in **.spec** format.

Look into the :ref:`SpectraContainer page <info_spectracontainer>` for a
detailed overview of the class.
